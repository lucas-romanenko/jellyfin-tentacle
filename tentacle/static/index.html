Install the Jellyfin Plugin

The Tentacle plugin adds a custom home screen and discover tab to Jellyfin. It's optional but recommended.

How to install:
  1. Open Jellyfin Dashboard → Plugins → Repositories
  2. Click + to add a new repository
  3. Paste the manifest URL below and name it Tentacle
  4. Go to Catalog, find Tentacle, click Install
  5. Restart Jellyfin to activate the plugin
Plugin Manifest URL
After installing, go to Jellyfin → Plugins → Tentacle → Settings and set the Tentacle URL to this server's address.
